# Scope

Project initialization and local development environment setup.

# Quick Start

Online playground: https://try.qingkuai.dev/

# Project Creation

```sh
# JavaScript version
npm create qingkuai -- my-app

# TypeScript version
npm create qingkuai -- my-app -ts
```

Supported package managers: npm, pnpm, yarn

# Development Server

```sh
cd project-directory
npm install && npm run dev
```
